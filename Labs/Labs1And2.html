Lab 1

Lab 1 is all about getting your environment set up. While this should be done before you arrive for the lab, we will be happy to help if you have had any issues. As a reminder, to complete the on-site labs, we recommend:

  • Visual Studio 2015 - Community Edition is fine
  • Azure SDK and Tools (with Azure Powershell)
  • Azure Account
If you are having trouble - you can always try using one of the Microsoft Virtual Machines - for instance A Virtual Machine with Azure SDK and Visual Studio 2015

Lab 2

Along the same lines, Lab 2 is all about setting up your initial account with Microsoft Azure.

If you have already done this, feel free to skip it and move on.
