Welcome to the 2016 Microsoft Azure Bootcamp.

InfoCraft and CODE Training are pleased to be able to present the North Houston instance of the Global Azure BootCamp. If you are in attendance today - you will find all the materials on this site. We will be going over most (maybe not all) of these over the course of the day. If you are NOT here in class for the big event, never fear. Contact one of the partner companies, and we will be happy to help you with your Azure and Cloud questions!

What will we be doing?

We will be following an agenda at the Azure Bootcamp. It was designed by the folks at Microsoft ™ for use at events just like this. We've tweaked some things just a bit to make sure the slides are up to date, so you will want to get your source from here if you are in attendance.

However the repo will not be actively maintained so you will likely want to visit to get fully up-to-date source in the future.
